# Phase 5 Handoff — Alpha AI Software Factory

Phase 5 connects Basalt's proof, graph, context, policy, transaction, and Command Center layers into the first complete alpha software-factory loop.

## Release target

- Package: `2.4.0a1`
- Tag: `v2.4.0-alpha.1`
- Branch: `phase-5-ai-software-factory`

## Delivered systems

1. Basalt Obsidian design system and Command Center redesign
2. Product Brain
3. Prevention-First Engineering
4. Deterministic State Coordinator
5. Dependency-aware Epoch Planner and Patch Aggregator
6. Provider-neutral heterogeneous Model Router
7. Governed specialist-agent orchestration
8. Proof-backed staging and VERIFIED-only atomic assembly
9. Factory CLI, API, Command Center views, artifacts, tests, and CI gate

## Supported alpha templates

- `python-service`
- `fullstack-lite`

## Trust invariant

A product target is not assembled until the staged build has passed Basalt verification. Contradictory intent, stale state, lock conflicts, failed proof, and unsupported templates fail closed.

## Phase 6 boundary

Phase 6 should productionize this loop for private beta with durable projects, authentication, multi-repository workspaces, persistent jobs, remote Git integration, stronger language/template coverage, isolated build infrastructure, collaboration, and private deployment environments.
