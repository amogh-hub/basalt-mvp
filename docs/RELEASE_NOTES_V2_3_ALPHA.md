# Basalt v2.3.0 Alpha 1 — Command Center Platform

Basalt now has a local browser-based operating surface for repository truth.

## Highlights

- Proof, risk, graph, transaction, approval, and evidence views
- Interactive impact analysis and Context Compiler
- Governed approval and rollback controls
- Read-only default mode
- Localhost-only default bind
- Same-origin and action-token security
- Zero runtime web dependencies
- 96 automated tests

## Core principle

**The interface compresses internal complexity into plain truth without weakening governance.**
