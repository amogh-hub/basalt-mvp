# Basalt v2.5.0 Beta 1 — Private Beta Full Build System

Phase 6 turns Basalt's alpha factory into a persistent private-beta product foundation.

## Highlights

- users, teams, roles, projects, and activity history
- durable jobs with leases, retries, idempotency, and cancellation
- bounded isolated job workspaces
- local and OpenAI-compatible provider registry
- expanded API, web-app, and SaaS starter templates
- VERIFIED-only deployment packages with SHA-256 integrity
- preview/staging/production approval ledger
- promotion, restore, and rollback records
- Private Beta Command Center views and APIs
- official founder-provided Basalt wordmark integration
- 187 automated tests and 102 critical proof-matrix tests

## Boundaries

This beta is a local/private deployment suitable for controlled evaluation. Production cloud hosting, distributed execution, enterprise auth, billing, observability, and live deployment connectors are not claimed until Phase 7.
