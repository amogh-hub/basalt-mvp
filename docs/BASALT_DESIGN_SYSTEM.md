# Basalt Obsidian Design System

Basalt's interface must match the product name: dark, dense, quiet, grounded, and precise.

## Visual direction

- near-black background
- graphite and obsidian surfaces
- restrained steel-blue accents
- soft off-white typography
- status colours used only for actual state
- subtle borders, measured spacing, and minimal motion

## Prohibited patterns

- lime accents
- emojis as product UI
- cartoon agent identities
- decorative neon and fake AI glow
- external frontend assets by default
- excessive inline styles

The `basalt factory design-system` command writes tokens and runs the governance audit.
