# Basalt v2.2.0 Alpha 1 — Safe Fix Platform

Phase 3 introduces governed agent-assisted safe fixes.

## Highlights

- Patch proposals instead of direct agent mutation
- Capability-based Policy Kernel
- Human approval with one-time tokens
- Repository state compare-and-swap
- Atomic patch application and backup
- Full before/after proof comparison
- Automatic rollback on failed or regressive proof
- Loop Governor with bounded revisions
- Persistent transaction and audit artifacts
- 76 automated tests

## Boundary

This alpha governs externally produced diffs and narrow deterministic proof-hardening proposals. It is not yet the full autonomous multi-agent software factory.
